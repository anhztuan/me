How to Use the API Endpoint Manager

This application allows you to manage your API endpoints easily. Here's how to use it:

1. View Endpoints

All your API endpoints are listed in the Endpoints tab with their method, URL, and status.

2. Add New Endpoints

Click the "Add New Endpoint" button to create a new API endpoint. Fill in the required information.

3. Toggle Status

Click on the status badge to toggle between active and inactive states.

4. Delete Endpoints

Use the Delete button to remove an endpoint from the system.

5. Admin Panel

Access the Admin Panel to manage website information and perform advanced configurations.
